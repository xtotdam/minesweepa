Minetris
========

A minesweeper with a pattern feature

###Controls:

* `d16` -- touch D-16
* `d16f` -- set/remove flag on D-16
* `d16, e16, f16` -- touch D-16, E-16 and F-16

We don't care about spaces. `d16f = d 16 f = d16 f`.

### Dependencies

* `numpy`
